Tappy Pet Press Kit

Description

  • Tappy Pet is a game about collecting pets and keeping them happy.
  • Tap on pets to make them happy.
  • Happy pets earn coins over time that you can spend on new pets.

Created By

Pohung Chen

Based in San Francisco, California. Previous work includes Osmo, Leap Motion, and IGF Finalist Perspective.

Ben Weatherall

Based in Australia. Previous work includes block ur socks, Crossy Road, and Spy Mouse.

Release Date

May 26, 2016.

Platforms

iPhone. Android.

Price

Free. (Ad supported and in-app purchasing).
